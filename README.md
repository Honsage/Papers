### Формализованные модели данных (III семестр)

[Текст КР](/Курсовые%20работы/Формализованные%20модели%20данных%20(III%20семестр).pdf)

**Тема**: Проектирование модели данных системы управления логистикой

**Описание**: Работа представляет собой проектирование модели данных для управления логистикой. Проводится теоретический анализ предметной области транспортной логистики, анализ математических основ оптимизации, изучение реляционных СУБД на примере PostgreSQL, формирование запросов на языке SQL. Результатом работы является модель данных системы управления транспортной логистикой, которая включает в себя аспект структуры в виде
системы сущностей (таблиц) и связей между ними, аспект манипуляции в виде набора оптимизационных SQL-запросов, аспект целостности в виде ограничений на данные в таблицах.

### Фронтенд-разработка (IV семестр)

[Текст КР](/Курсовые%20работы/Фронтенд-разработка%20(IV%20семестр).pdf)

[Исходный код проекта](https://github.com/Honsage/Didacticum)

**Тема**: Приложение для создания интерактивных учебных материалов

**Описание**: Работа представляет собой проектирование, разработку и тестирование фронтенда веб-платформы для создания, публикации и распространения интерактивных учебных материалов. В рамках работы был проведен анализ предметной области и составлена концепция продукта. Сформированы требования к продукту и его проект. На основе проектирования произведены реализация и тестирование програмнного кода. Результатом работы является фронтенд-часть веб-платформы для создания интерактивных учебных материалов, а также концпеция и бренд продукта.

[Публикация](https://elibrary.ru/item.asp?id=80384541)

**Описание**: В работе исследуется приложение функции Ламберта к решению некоторых трансцендентных уравнений, возникающих в различных областях математики и физики.
